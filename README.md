## Poll Tracer

### Installation

1. [Drag me to your bookmarks bar](javascript:void(function(){var s=document.createElement('script');s.src='https://raw.github.com/intentmedia/poll_tracer/master/poll_tracer.js';document.body.appendChild(s);})())

2. Click on any page to see what, if anything, is using setTimeout or setInterval.

### Warning

This will effectively stop the page from functioning if it relies on polling, as it captures and thereby prevents setTimeout/setInterval call execution.
